# 🎬 CinemaTix

CinemaTix is a Flutter-based movie ticket booking application developed as a university project.

## ✨ Features

- User Login & Register
- Admin Dashboard
- Manage Movies (Add, Edit, Delete)
- Movie Search
- Favorite Movies
- Movie Details
- Seat Selection
- Multiple Payment Methods
- QR Code E-Ticket
- Booking History
- Admin Booking Management
- User Management

## 🛠 Technologies

- Flutter
- Dart
- Material 3
- qr_flutter
- url_launcher

## 📂 Project Structure

```text
lib/
├── main.dart
assets/
├── posters/
├── backdrops/
```

## 🚀 Getting Started

1. Clone this repository

```bash
git clone https://github.com/Arjuna401/CinemaTix.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the application

```bash
flutter run
```
